# Dự Án Quản Lý Dịch Vụ Du Lịch

Đây là mã nguồn của đề tài quẩn lý dịch vụ du lịch trong môn lập trình hướng đối tượng MI4090 mã lớp 163629 của nhóm 201.

## Công nghệ sử dụng
* Ngôn ngữ: C# (WinForms/WPF/ASP.NET)
* Cơ sở dữ liệu: SQL Lite
#Lưu ý:
  Có một file Data mẫu DATA.DB ở đầu.
  Khi không có dữ liệu ban đầu thì ứng dụng mặc định tạo 1 tài khoản admin có username: admin và password: admin
  File DATA.DB sẽ lưu database của chương trình và được khởi tạo cùng folder với file .exe
