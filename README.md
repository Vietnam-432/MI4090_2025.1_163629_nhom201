# Dự Án Quản Lý Du Lịch

Đây là phần mềm quản lý thông tin khách du lịch, tour và đặt chỗ được phát triển bằng C# và Visual Studio.

## Tính năng chính
* Quản lý danh sách tour du lịch.
* Quản lý thông tin khách hàng.
* Thống kê doanh thu theo tháng.

## Công nghệ sử dụng
* Ngôn ngữ: C# (WinForms/WPF/ASP.NET)
* Cơ sở dữ liệu: SQL Server
* Công cụ: Visual Studio 2022

## Hướng dẫn cài đặt
1. Clone dự án: `git clone <link-github-cua-ban>`
2. Mở file `.sln` bằng Visual Studio.
3. Cấu hình chuỗi kết nối Database trong file `app.config`.
4. Nhấn **F5** để chạy dự án.
